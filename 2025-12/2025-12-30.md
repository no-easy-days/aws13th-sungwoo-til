## 📝 배운 내용 (Today I Learned)

### 변수
파이썬에서는 다중 할당 지원
다중 할당: 여러 변수에 동시에 값을 할당
```python
first = "A"
second = "B"

first, second = second, first

print(first)   # 출력: B
print(second)  # 출력: A
```

### class
기본 형태
```python
class 클래스명(self, 요소1, 요소2, ...)
  def __init__(self, 매개1, 매개2, ...)
  self.매개1 = 매개1
  self.매개2 = 매개2
  ...

  def 함수명(self, 매개, ...):

변수 = 클래스명()
변수.함수명(인자1, 인자2, ...)
```

코드 짤 때 쉽게 할 수 있는 방법만 생각하지 말자 = 하나하나 동작하는 것을 부정적으로 생각하지 말자.

### enumerate
- 인덱스와 원소를 동시에 가져올 수 있음
```python
    def show_tasks(self):
        for index, task in enumerate(self.tasks, 1): #1은 시작 숫자
            print(f"{index}번째: {task}")
```
