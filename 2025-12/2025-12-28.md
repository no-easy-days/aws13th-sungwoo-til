### 변수와 자료형(int, float, str, bool, list, dict, tuple, set)

변수
- 컴퓨터가 메모리에 값을 저장해 놓은 위치 이름
- 그 값은 변할 수 있으므로 "변할 수 있는 수(데이터)"-> "변수"
- 값이 변해도 수정 편리

int: 음의 자연수, 0, 양의 자연수
float: 정수 + 소수점 이하의 값도 포함
str: 연속된 문자
bool: 값이 같으면 True, 다르면 False
list: 여러 개의 값이나 변수를 한꺼번에 담기 가능, 음수 인덱스도 사용 가능
```python
n_list = [11, 22, 33, 44, 55]

n_list[1:3]
```
dict: 키와 값을 쌍으로 갖는 항목이 나열된 자료형, 키가 중복되면 안됨
```python
person = {'이름' : '홍길동', '나이' : '26', '몸무게' : '82'}
person['이름']
```
tuple: 한 번 정해진 요소의 순서를 바꿀 수 없는 컬렉션 자료형, 여러개의 객체 반환 가능
```python
t0 = (10, 20, 30)
t1 = t0 + t0
t1
```
set: 주어진 기준을 만족시키는 대상의 모음, 중복 없음, 순서가 없는 원소의 모임이므로 인덱스 사용 불가 => 슬라이싱 불가
```python
s = {100, 100, 200, 200, 300, 400}
s
```

### 제어문(if, for, while)

if
- 특정한 조건에 따라 실행 내용이 달라지는 문장 형태
- 들여쓰기로 실행될 코드 덩어리를 나눔
- if: 만약 ~하면
- else: 그 이외에
- elif: 그 이외일 때 만약 ~하면
```python
score = 90

if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")
```

for
- 같은 문장을 여러 번 반복해야 할 때 효율적이며 코드 가시성 굿
- range: 특정 구간의 정수 열을 생성, 반복문에서는 생성 값이 차례대로 할당
- 중첩 for문: for문 안에서 실행되는 for문
```python
for i in range(2, 10):
  for j in range(1, 10):
    print('{}*{} = {:2d}'.format(i, j, i*j), end = ' ')
  print()
```

while
-이 참인 동안 계속 실행되는 반복문
- 수행 횟수를 정확하게 모르지만 수행 조건이 명확한 경우에 적합 ex)가위바위보
```python
count = 3

# "카운트가 0보다 큰 동안 계속 실행해"
while count > 0:
    print(f"카운트다운: {count}")
    count = count - 1  # 1씩 줄임

print("발사! 🚀")
```

### 함수 정의와 호출

함수
- 미리 만들어 놓고 호출하여 큰 프로그램을 쉽게 짜게 해줌
- 내장함수: 파이썬에서 미리 만들어 제공
- 사용자 정의 함수: 사용자가 직접 필요한 함수 제작
- 인자: 함수 외부에서 넘겨주는 값
- 매개변수: 함수 내부에서 사용되는 변수

```python
def get_sum(a,b):
  return a+b

result = get_sum(100, 200)
print('두 수의 합', result)
```
