## 📝 배운 내용 (Today I Learned)

### curl 명령어
- HTTP 요청을 직접 보낼 수 있음
- 옵션
  - -X: HTTP 메서드 지정
  - -H: 헤더 추가
  - -d: 요청 본문 데이터
  - -v: 상세 출력(디버깅용)
  - -i: 응답 헤더 포함 출력

### 주요 Request Header 종류
- `Host`: 요청 받을 서버 주소
- `content-type`: 데이터 형식 명시
- `Authorization`: 인증 정보 전달
- `Set-cookie`: 브라우저에 쿠키 저장 지시
