## VPC(Virtual Private Cloud=가상 사설 클라우드)

- 정의: 클라우드 환경에서 제공되는 나만의 가상 네트워크 공간
- 이 공간은 완전히 격리되어 있어 외부 사용자가 직접 접근할 수 없고 내가 원하는대로 네크워크 설정 가능
- 설정 가능한 것들
    - IP 주소 대역, 서브넷, 라우팅 테이블, 게이트웨이 등
    - 방화벽 규칙을 사용해 외부에서 접근할 수 있는 범위 제한
    - 필요에 따라 네트워크를 확장 혹은 축소
- 필요성
    - 보안 - 가장 큰 이유는 데이터를 안전하게 중요하게 보호(프라이빗 한 공간에 안보이게 VPC로 감싸버림)
    - 확장성 - 가상이기 때문에 사용자가 필요할 때마다 자원을 쉽게 늘렸다 줄일 수 있다 ex) 트래픽 피크타임
    - 유연한 네트워크 설계 - 다양한 기업의 요구에 맞게 커스터마이징 가능

## 로드 밸런서

- 정의: 서버에 걸리는 부하(트래픽)을 여러 서버에 균등하게 분배
- 역할
    - 트래픽 분배 - 요청을 처리할 수 있는 서버들 중 하나로 트래픽 분베
    - 장애 허용 - 특정 서버에 장애가 생겨도 로드 밸런서가 정상 서버로 요청을 보냄
    - 확장성 - 서버 수를 늘려도 로드 밸런서가 자동으로 새 서버를 트래픽 분배에 포함시킴
    - 성능 최적화 - 각 서버의 상태를 주기적으로 확인하여 과부하 상태에 있는 서버로 요청을 보내지 않음
- 작동방식
    - Round Robin → 순차적으로 서버에 요청을 분배
    - Least Connection → 현재 연결된 사용자가 가장 적은 서버로 요청 보냄
    - Weighted → 성능이 좋은 서버에 더 많은 요청을 보내도록 설정
    - IP 해시 기반, 지리적 위치 기반 등등
- 상태 확인: 주기적으로 서버의 상태를 확인
- 종류
    - 하드웨어 로드밸런서 - 물리적 장비로 구성, 성능과 안정성이 높지만 비용이 높음
    - 소프트웨어 로드밸런서 - 소프트웨어로 트래픽 분배, 비용 효율적이고 클라우드 환경에서 자주 사용

## 인터네트워킹

- 정의: 크지 않은 여러 개의 네트워크를 묶어서 늘려서 네트워크 규모 확장
- 우리가 평소에 사용하는 인터넷도 “인터네트워킹”으로 묶여 있음
- 인터네트워크 통신 수단: TCP/IP

## 프로토콜

- 정의: 컴퓨터끼리나 컴퓨터와 기계가 통신할 때 어떤 데이터를 어떠한 순서로 주고 받을지 정해둔 것
- 통신 프로토콜
    - 데이터 형식: 주고 받는 데이터가 어떤 형식인지 정함
    - 통신 수단: 어떤 순서로 무엇을 주고 받을지 정함

## IP

- 인터넷 상에서 통신 상대를 특정하는 식별자가 IP 주소
- 총 32비트(8 8 8 8), 10진수 표기법 사용(0~255)
    - 32비트에서 왼쪽부터 24비트는 네트워크 주소(네트워크를 특정하는 정보 저장)
    - 나머지 비트는 호스트 주소(어떤 네트워크 안에서 해당하는 컴퓨터를 특정하는 정보)
    - 이 두가지 정보로 IP 주소 하나만으로 어떤 네트워크에 속한 컴퓨터인지 특정 가능

## IP 주소 클래스

- 네트워크 주소와 호스트 주소를 나누는 위치를 변경하는 것도 가능
- 네트워크 주소 길이에 따라
    - 8비트 → A 클래스
    - 16비트 → B 클래스
    - 24비트 → C 클래스
- 네트워크 주소 길이와 호스트 주소 길이가 반비례하므로 네트워크 주소 길이가 짧을수록 그만큼 식별할 수 있는 컴퓨터 증가
- 호스트의 비트가 모두 0이면 네트워크 자체, 모든 비트가 1이면 브로드캐스트 주소

## Docstring

- 정의: 문서화 문자열 리터럴
- 용도: 코드 문법 설명이 아닌 대상을 사용할 개발자를 위한 설명서
- 일반 주석과 달리 실행 중 참조할 수 있는 객체
- 작성 위치와 문서화 대상
    - 모듈: 파일 상단의 설명
    - 클래스: 클래스의 목적 및 속성 설명
    - 함수: 함수의 인자, 반환값 설명
    - 메서드: 클래스 내부 함수 설명
- 좋은 dicstring의 내용
    - 설명대상에 대한 요약 내용 기재, 상세한 설명이 이후 추가
    - 상세 설명 이후 간단한 사용 예제 기재
    - 명령형 문장 사용
    - 함수 인자, 반환값, 예외 명시
- 스타일
    - 스핑크스 - 자세한 설명 제공
        
        `class Vehicle(object):
            '''
            The Vehicle object contains lots of vehicles
            :param arg: The arg is used for ...
            :type arg: str
            :param `*args`: The variable arguments are used for ...
            :param `**kwargs`: The keyword arguments are used for ...
            :ivar arg: This is where we store arg
            :vartype arg: str
            '''`
        
        `def __init__(self, arg, *args, **kwargs):
                self.arg = arg`
        
        `def cars(self, distance, destination):
